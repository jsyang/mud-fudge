# mud-fudge
MD-FDG - MarkDown Force Directed Graph

Sample FDG layout provided (BikeCrypt)

performance in mind. Adapted from [github.com/kennethkufluk/js-mindmap](https://github.com/kennethkufluk/js-mindmap)
